# RabbitGPT

RabbitGPT is a beautiful frontend for CHATGPT

## Getting Started

First, run the development server:

```bash
yarn install
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Learn More

RabbitGPT is made with NEXT.JS , TailwindCSS
